# SOC Alert Triage Playbook (SOC L1)

This repository demonstrates real-world SOC Level 1 alert triage workflows.
It focuses on alert validation, investigation steps, false-positive analysis,
MITRE ATT&CK mapping, and escalation decisions.

## Alerts Covered
- Brute Force Login Attempts
- Malware / Suspicious Process Execution
- Phishing Email Attacks
- Suspicious Login / Impossible Travel

## Skills Demonstrated
- SOC L1 alert triage
- Log analysis
- Incident investigation
- MITRE ATT&CK mapping
- Security documentation
